# Cognifyz Internship - Level 2 Tasks

This repository contains Java programs completed as part of the Cognifyz Technologies Internship.

## Level 2 - Task 1: Tic-Tac-Toe Game

A two-player Tic-Tac-Toe game developed in Java.
The program displays a 3x3 game board, allows both players to enter their moves, checks winning conditions, and displays the final result.

### Skills Used

* Arrays
* Loops
* Conditional statements
* User input handling

## Level 2 - Task 2: Password Strength Checker

A Java program that checks the strength of a password entered by the user.
It analyzes the password based on length, uppercase letters, lowercase letters, digits, and special characters.

### Skills Used

* String manipulation
* Conditional statements
* User input handling

## Level 2 - Task 3: File Encryption and Decryption

A Java program that encrypts and decrypts the content of a text file using a simple encryption technique.
The user can choose encryption or decryption and provide input and output file paths.

### Skills Used

* File handling
* String manipulation
* Input/output operations
